## 🎓 毕业级双轨实战项目

### 🛠️ 项目 A：前道 CVD 工艺设备控制系统模拟
* **要求**：使用 C++/Qt 20 开发，包含 Linux 实时控制层与 HIL 仿真
* **功能清单**：
  1. **多腔体状态机系统**：模拟 Process Chamber / Load Lock / TM 的独立与联动状态流转（含握手协议）
  2. **Recipe 执行引擎**：解析 JSON 格式工艺菜单，实现带 PID 反馈的虚拟气体/真空闭环控制
  3. **Interlock 保护系统**：如真空未达标禁止开启挡板阀，硬件异常触发急停，支持安全状态自动切换
  4. **SECS/GEM 模拟接口**：实现 S6F11 事件上报、S2F21 远程启停，并对接 GEM300 (E87/E40/E94) 流程
  5. **高级功能**：集成基础 FDC 监测系统与 R2R 反馈逻辑模块

### 🛠️ 项目 B：后道 Wafer Sort 自动测试平台
* **要求**：使用 Python/C#/大数据工具链 开发，集成 32-Site 并行仿真
* **功能清单**：
  1. **虚拟 Prober 控制**：生成 300mm 晶圆坐标系，控制 XY 走位，模拟 Z 轴接触力反馈与清针时序
  2. **虚拟 ATE 引擎**：模拟输出测试电压/电流，加入随机硬件失效分布，支持多 Site 资源竞争模拟
  3. **测试程序执行**：跑通 Test Flow，根据判定阈值进行 Die 级别判定，支持 Pattern 动态加载
  4. **数据生成与可视化**：生成标准二进制 STDF 文件，输出 Wafer Map 并在看板实时显示良率统计
  5. **智能化分析**：基于 Wafer Map 自动识别典型的光刻/刻蚀系统性失效模式（如边缘失效/聚类失效）
