# 📘 半导体设备全景介绍（硬件 · 结构 · 软件 · 算法）

> **导读**：本目录按制造流程顺序，系统介绍半导体制造中涉及的所有核心设备。每类设备从 **硬件结构** → **软件架构** → **核心算法** 三个维度展开，帮助软件工程师理解"代码控制的到底是什么"。

## A. 硅片制造设备（Wafer Manufacturing）

### A.1 单晶硅生长炉（Czochralski / Float Zone Crystal Puller）
* **硬件结构**
  * 石英坩埚、石墨加热器、籽晶夹持机构、旋转提升装置
  * 气氛保护系统（Ar/N₂）、冷却水循环、真空/常压切换模块
  * 称重传感器（熔硅重量）、温度传感器（多点热电偶 + 红外高温计）
* **软件架构**
  * 坩埚旋转 + 籽晶提升双轴运动控制（速度精度 ±0.01 mm/min）
  * PID 温度闭环控制（1420°C ±0.5°C）
  * 直径控制（CCD 视觉 + 重量变化率反馈）
* **核心算法**
  * 直径预测与生长速率控制：基于固液界面热场模型的 PID + 前馈补偿
  * 旋转速度优化：抑制氧/碳浓度径向不均匀性
  * 缩颈（Neck）工艺自动控制：位错密度最小化

### A.2 晶圆切割机（Wafer Slicing / Wire Saw）
* **硬件结构**
  * 金刚石线锯（线径 80~120μm）、砂浆供给系统、张力控制机构
  * 晶棒进给平台（高精度滚珠丝杠）、冷却液循环过滤
* **软件架构**
  * 多线张力独立闭环控制（每根线独立张力传感器）
  * 切割速度与进给速率协同控制
  * 断线检测与自动停机保护
* **核心算法**
  * 张力-进给耦合控制：防止线弓（Wire Bow）导致切割面弯曲
  * 切割路径优化：减少总切割时间同时保证 TTV（Total Thickness Variation）

### A.3 晶圆研磨/抛光设备（Lapping & CMP）
* **硬件结构**
  * 上下研磨盘（双面研磨）、研磨液供给泵、厚度测量模块
  * 载具（Carrier）、配重块、清洗/干燥模块
* **软件架构**
  * 压力-转速-时间三参数闭环控制
  * 在线厚度测量（光学/电容式）与终点检测
  * 研磨液流量自动调节
* **核心算法**
  * Preston 方程应用：去除速率 ∝ 压力 × 相对速度
  * 非均匀性补偿：通过边缘/中心分区加压改善 WIWNU

## B. 薄膜沉积设备（Deposition）

### B.1 化学气相沉积设备（CVD / PECVD / LPCVD / MOCVD）
* **硬件结构**
  * **反应腔体**：石英/铝/不锈钢腔体，分区加热（最高 1200°C）
  * **气体系统**：MFC（质量流量控制器）×N路、气路阀门矩阵（VMB）、前驱体鼓泡器
  * **真空系统**：干泵 + 分子泵、蝶阀/闸板阀、真空计（皮拉尼/电容薄膜）
  * **RF 射频系统**：13.56MHz/27.12MHz 射频电源 + 阻抗匹配网络 + 等离子体探针
  * **排气系统**：尾气处理（Burn/Wet Scrubber）、颗粒过滤器
* **软件架构**
  * **主控状态机**：IDLE → LOAD → PURGE → PROCESS → UNLOAD → ERROR
  * **Recipe 执行引擎**：Step-level 控制（气体流量 / RF功率 / 压力 / 温度 / 时间）
  * **真空联锁**：泵启动顺序、阀门开关互锁、泄漏检测
  * **多腔体调度**：Cluster Tool 路由（Load Lock ↔ Transfer Chamber ↔ Process Chamber）
  * **SECS/GEM 接口**：S6F11 CEID 上报、S2F41 远程命令
* **核心算法**
  * **压力 PID 控制**：蝶阀开度与泵速解耦，处理从 760 Torr 到 mTorr 的宽范围
  * **温度多区 PID 解耦**：消除加热区之间的热耦合干扰
  * **RF 阻抗匹配**：Smith 圆图自动调谐，50Ω 匹配精度
  * **均匀性优化**：Gas Flow Pattern + Showerhead 孔径分布优化
  * **终点检测**：光学发射光谱（OES）实时监控膜厚

### B.2 物理气相沉积设备（PVD / Sputtering）
* **硬件结构**
  * 靶材（Target）：旋转靶/平面靶，磁控管阵列
  * 晶圆台：可旋转 + 加热（最高 500°C）+ 偏压（RF/DC）
  * 挡板（Shutter）、屏蔽罩（Shield）、低温冷阱（Cryopump）
* **软件架构**
  * 溅射功率控制（DC/RF/Pulsed DC）
  * Ar 气流量与腔室压力闭环
  * 靶材寿命监测与换靶提醒
* **核心算法**
  * 靶材侵蚀轮廓预测：磁场分布 → 薄膜厚度均匀性
  * 沉积速率补偿：根据靶材消耗自动调整功率/时间
  * 薄膜应力控制：基板偏压与工作气压的联合优化

### B.3 原子层沉积设备（ALD）
* **硬件结构**
  * 前驱体脉冲阀（μs 级响应）、吹扫气体系统
  * 加热基座（150~400°C）、原位膜厚测量（椭偏仪）
* **软件架构**
  * **脉冲时序控制**：Precursor Pulse → Purge → Reactant Pulse → Purge，周期精度 ±1ms
  * 循环次数计数与累积厚度估算
  * 前驱体温度/压力监控
* **核心算法**
  * 自饱和检测：通过 QMS（四极质谱）判断前驱体吸附饱和点
  * GPC（Growth Per Cycle）在线标定
  * 高深宽比保形性（Conformality）优化

## C. 刻蚀设备（Etching）

### C.1 等离子体干法刻蚀设备（Plasma Etch / ICP / CCP / RIE）
* **硬件结构**
  * **等离子体源**：ICP（电感耦合）/ CCP（电容耦合）线圈/平板
  * **反应腔**：铝/硅/石英腔体，静电卡盘（ESC）冷却
  * **气体系统**：腐蚀性气体（Cl₂/BCl₃/SF₆/CF₄/O₂）多路供给
  * **Endpoint 检测**：OES（光学发射光谱）、激光干涉仪
  * **真空系统**：涡轮分子泵 + 干泵、冷阱
* **软件架构**
  * 多参数 Recipe 执行：气体流量 / RF功率 / 压力 / 基板温度 / 偏压
  * Endpoint 信号实时处理与刻蚀终止
  * 腔室条件匹配（Chamber Matching）：不同腔体间的刻蚀速率补偿
* **核心算法**
  * **刻蚀选择比控制**：通过气体比例调节 SiO₂/SiN/Si 选择比
  * **Profile（侧壁形貌）控制**：离子能量/角度分布建模
  * **CD（关键尺寸）均匀性**：Showerhead + 边缘环（Edge Ring）联合优化
  * **等离子体密度均匀性**：ICP 线圈分层功率分配
  * **OES Endpoint 算法**：特征波长强度微分检测

### C.2 湿法刻蚀/清洗设备（Wet Etch / Wet Clean）
* **硬件结构**
  * 槽式/单片式结构、化学品供给泵（HF/H₂O₂/NH₄OH/H₂SO₄）
  * 温控水浴/化学品加热、超声波发生器（Megasonic）
  * 排液/废液收集系统
* **软件架构**
  * 浸泡时间精确控制（±0.1s）
  * 化学品浓度在线监测（比重/电导率）
  * 溢流/喷淋模式切换
* **核心算法**
  * 蚀刻速率温度补偿（Arrhenius 方程）
  * 洗涤效率（SE）优化：频率 × 功率 × 时间的多目标优化

## D. 光刻设备（Photolithography）

### D.1 光刻机（Scanner / Stepper — DUV / EUV）
* **硬件结构**
  * **光源系统**：ArF（193nm）准分子激光器 / EUV（13.5nm）等离子体光源 + 多层膜聚光镜
  * **照明系统**：自由曲面反射镜、DOE（衍射光学元件）、光阑
  * **掩模台（Reticle Stage）**：磁悬浮/气浮平台，加速度 >10g，定位精度 <1nm
  * **晶圆台（Wafer Stage）**：六自由度纳米定位台，双台交替曝光，速度 >2m/s
  * **投影物镜**：反射式/折射式，NA 0.33(EUV)/1.35(DUV)，面型精度 <0.1nm RMS
  * **对准系统**：TTL（Through The Lens）激光对准，套刻精度 <1.5nm
  * **环境控制**：恒温 ±0.01°C、气压 ±0.1%、振动 <0.5nm
* **软件架构**
  * **实时运动控制**：μm 级台面控制，延迟 <10μs
  * **曝光剂量控制**：激光脉冲能量反馈 + Dose 均匀性补偿
  * **对准与套刻**：Mark 检测 → 偏移计算 → 坐标变换 → 台面补偿
  * **EUV 光源功率管理**：锡液滴发生器 + 激光打靶同步
  * **产能调度**：双台 Tandem 模式，Load → Align → Expose 流水线
* **核心算法**
  * **套刻补偿（Overlay Correction）**：高阶多项式（如 37 参数模型）拟合
  * **OPC（光学邻近修正）**：掩模图形逆向优化，Sub-Resolution Assist Feature 生成
  * **EUV 光源功率稳定性控制**：锡滴位置/速度反馈 + 激光脉冲同步
  * **剂量均匀性算法**：脉冲能量归一化 + 扫描速度自适应调节
  * **套刻测量**：图像处理 + 互相关算法，精度 <0.2nm

### D.2 电子束光刻设备（E-Beam Lithography）
* **硬件结构**
  * 电子枪（热发射/LaB₆/场发射）、电磁透镜系统、偏转系统
  * 芯片台（激光干涉仪定位）、真空系统
* **软件架构**
  * 图形数据流处理（GDSII → 逐像素曝光序列）
  * 电子束偏转与台面移动协同
  * 近邻效应校正（PEC）
* **核心算法**
  * 空间电荷效应模拟
  * 电子散射点扩展函数（PSF）卷积
  * 剂量邻近校正

## E. 离子注入设备（Ion Implantation）

### E.1 离子注入机（Medium / High Current / High Energy）
* **硬件结构**
  * **离子源**：RF/微波等离子体源，产生活性气体等离子体
  * **质量分析磁铁**：扇形磁铁，选择特定 m/q 比离子
  * **加速管/减速器**：能量范围 100eV ~ 60MeV
  * **扫描系统**：静电偏转 / 机械扫描 / 双扫描
  * **Faraday 束流测量**：法拉第杯阵列 + 电荷积分
  * **晶圆台**：高温（最高 800°C）/ 低温（-120°C）选配
* **软件架构**
  * 剂量控制：束流强度 × 时间积分，精度 <0.5%
  * 能量控制：加速电压闭环
  * 角度控制：倾斜/旋转角度精确定位
  * 均匀性扫描参数优化
* **核心算法**
  * **剂量均匀性**：机械扫描速度分布优化 + 三角波/梯形波扫描波形
  * **束流衰减补偿**：离子源老化引起的束流漂移自动调节
  * **SIMS（二次离子质谱）数据拟合**：LSS 理论 + Monte Carlo 注入轮廓

## F. 热处理设备（Thermal Processing）

### F.1 扩散/氧化炉（Diffusion / Oxidation Furnace）
* **硬件结构**
  * 水平/垂直石英管（单管/三管/五管）
  * 电阻加热器（SiC/石墨），最高 1200°C
  * 石英舟/碳化硅舟（载片 150~300 片）
  * 气体系统：O₂/N₂/H₂/HCl/N₂O
* **软件架构**
  * 温区多点 PID 控制（1~5 区）
  * 气体时序控制（多步 Recipe）
  * 舟进出自动化（机械臂/链条传送）
* **核心算法**
  * 扩散模型：Fick 定律数值求解 → 时间-温度曲线优化
  * 氧化速率模型：Deal-Grove 模型在线修正
  * 热预算（Thermal Budget）最小化

### F.2 快速热处理设备（RTP / RTA / Spike Anneal）
* **硬件结构**
  * 灯管阵列（卤素灯/氙灯），升温速率 >200°C/s
  * 反射腔体（减少热辐射损失）
  * 光学 pyrometer 非接触测温
* **软件架构**
  * 超快速温度闭环（采样率 >1kHz）
  * 脉冲灯管功率独立控制
  * 峰值温度 + 总时间（Thermal Budget）双重约束
* **核心算法**
  * **多区温度均匀性**：灯管分区功率分配自适应算法
  * **温度曲线跟踪**：前馈 + PID 组合，过冲 <±2°C
  * **Spike Anneal 控制**：升温速率 >300°C/s，保温 <1s

## G. 化学机械抛光设备（CMP）

* **硬件结构**
  * 抛光盘（Carrier）+ 研磨垫（Pad）+ 研磨液（Slurry）供给
  * 下压力系统（气膜/液压分区加压）

* **核心算法**
  * **Preston 方程 + 扩展模型**：去除速率预测
  * **WIWNU 优化**：边缘/中心分区压力实时调节
  * **碟陷（Dishing）/ 凹陷（Erosion）补偿**：图案密度感知压力分布

## H. 量测与检测设备（Metrology & Inspection）

### H.1 光学量测设备（Optical Metrology）
* **硬件结构**
  * 光谱椭偏仪（SE）、反射光度计、散射仪
  * 宽光谱光源（DUV~IR）、高精度光谱仪
  * 高精度 XY 台（精度 <10nm）
* **软件架构**
  * 光谱数据采集与预处理（去噪、归一化）
  * 库搜索 + 逆向拟合实时反馈
* **核心算法**
  * **多层薄膜建模**：Fresnel 方程 + 传输矩阵法
  * **逆问题求解**：Levenberg-Marquardt 拟合 → 膜厚/折射率
  * **CD-Semu（散射测量逆向）**：严格耦合波分析（RCWA）

### H.2 电子束检测/量测设备（E-Beam Inspection / CD-SEM）
* **硬件结构**
  * 热场/冷场发射电子枪、电磁透镜、探测器阵列
  * 精密样品台（激光干涉仪）
* **软件架构**
  * 高速电子束扫描控制
  * 信号采集与实时图像处理
  * 自动缺陷分类（ADC）
* **核心算法**
  * **缺陷检测**：Die-to-Die / Die-to-Database 差异比较
  * **CD 测量**：边缘检测 + 统计拟合
  * **ADC 分类**：CNN/ResNet 缺陷类型自动分类

### H.3 薄膜/缺陷检测设备（Film / Defect Inspection）
* **硬件结构**
  * 激光散射暗场检测（如 KLA SP 系列）
  * 明场/暗场光学系统
  * 高速数据采集卡（GB/s 级）
* **软件架构**
  * 全晶圆扫描路径规划
  * 实时散射信号处理与阈值判定
  * 缺陷地图生成与聚类分析
* **核心算法**
  * **信噪比优化**：背景扣除 + 多帧平均 + 自适应阈值
  * **系统性缺陷识别**：Wafer Map 空间聚类（DBSCAN/K-Means）
  * **假缺陷（Nuisance）过滤**：基于特征的机器学习分类

## I. 测试设备（Test / ATE）

### I.1 晶圆级测试设备（Wafer Sort / CP Test）
* **硬件结构**
  * **ATE 主机**：高速数字通道（>1GHz）、精密模拟源表（SMU）、DMM
  * **探针台（Prober）**：XYθZ 五轴精密平台，触针（Contactor）阵列
  * **温控模块**：热台/冷台（-65°C ~ 300°C）
* **软件架构**
  * Test Program 执行引擎（测试流程 + 条件分支）
  * Prober ↔ ATE 握手协议（SEMIF7）
  * 多 Site 并行测试调度
  * STDF 数据生成与实时传输
* **核心算法**
  * **多 Site 资源竞争调度**：共享通道分配 + 优先级仲裁
  * **自适应测试**：基于历史数据的测试项动态跳过
  * **Wafer Map 统计分析**：缺陷模式识别 + 良率趋势预警

### I.2 成品测试设备（Final Test / FT）
* **硬件结构**
  * **分选机（Handler）**：Pick-and-Place / 转塔式，吞吐量 >30K UPH
  * **温控腔**：多温度同时测试（Hot + Cold + Room）
  * **ATE 通道数**：最高 >1000 通道
* **软件架构**
  * Bin 分类与分选逻辑
  * Handler ↔ ATE 时序同步（TRAY/SINGULATE 信号）
  * 并行测试资源分配
* **核心算法**
  * **Test Flow 优化**：基于先验信息的排序（快速失败 Fast-Fail）
  * **多温区并行调度**：冷/热/室温槽同步测试
  * **假失效（Retest）策略**：概率模型决定是否复测

## J. 封装设备（Packaging）

### J.1 引线键合设备（Wire Bonder）
* **硬件结构**
  * 超声波楔焊/球焊头（40~125kHz）
  * XYθZ 高速运动平台（加速度 >100g）
  * 金线/铜线/铝线供给（线径 15~50μm）
  * 线尾火焰球成型（EFO）
* **软件架构**
  * 运动轨迹实时插补（Look-Ahead 路径规划）

  * **线弧高度控制**：基于拉力测试数据的参数自学习

### J.2 倒装芯片键合设备（Flip-Chip Bonder）

* **核心算法**

  * **共面性（Coplanarity）补偿**：多点高度测量 + 差异化压力

### J.3 塑封设备（Molding / Transfer Molding）

  * 载荷任务分配（Dispatching）

## M. 设备核心通用模块（Cross-cutting Subsystems）

### M.1 真空系统（Vacuum Subsystem）
* **硬件**：干泵（50~3000 m³/h）、分子泵（<10⁻⁹ Torr）、低温泵、各类阀门、真空计
* **软件**：泵启动/停止序列、联锁逻辑、泄漏检测、压力 PID
* **算法**：真空建模（Monte Carlo 分子流模拟）、泵速衰减预测

### M.2 气体系统（Gas Delivery）
* **硬件**：MFC（0~10000 sccm）、VMB（Valve Manifold Box）、气体盘、特气柜
* **软件**：气体浓度/流量闭环、吹扫序列、毒性气体泄漏检测
* **算法**：MFC 自动归零、混合气比例 PID、死区容积吹扫时间计算

### M.3 温控系统（Thermal Control）
* **硬件**：电阻加热器、冷却器（水/氟化液）、热电偶、红外测温
* **软件**：多区 PID、升温/降温 Profile 执行、热预算监控
* **算法**：PID 自整定、Smith 预估器补偿大滞后、模型预测控制（MPC）

### M.4 运动控制系统（Motion Control）
* **硬件**：直线电机/音圈电机、光栅尺/激光干涉仪、压电陶瓷平台
* **软件**：实时插补、PID 位置/速度环、电子凸轮
* **算法**：S-Curve / 梯形加减速、振动抑制（输入整形 Input Shaping）、前馈补偿

### M.5 射频/等离子体系统（RF / Plasma）
* **硬件**：射频电源（13.56/27.12/60/40.68 MHz）、匹配器、等离子体探针
* **软件**：功率闭环、匹配调谐、等离子体启停序列
* **算法**：阻抗匹配 PID、等离子体状态分类（OES + 机器学习）

### M.6 视觉对准系统（Vision / Alignment）
* **硬件**：CCD/CMOS 相机、远心镜头、同轴/环形光源
* **软件**：图像预处理、模板匹配、坐标变换
* **算法**：亚像素边缘检测、仿射变换标定、Notch/Fiducial 搜索
